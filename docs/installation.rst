============
Installation
============


Stable release
--------------

To install bollywood-data-science, run this command in your terminal:

.. code-block:: console

    $ pip install bollywood-data-science

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


Dev build
---------

The sources for bollywood-data-science can be downloaded from the `Github repo`_.

These sources can be installed with `Poetry`_

.. code-block:: console

    $ git clone git@github.com:charmoniumQ/bollywood-data-science.git
    $ cd bollywood-data-science
    $ pip install poetry
    $ poetry install

.. _Github repo: git@github.com:charmoniumQ/bollywood-data-science.git
.. _Poetry: https://python-poetry.org/
